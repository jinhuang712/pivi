# 全局快捷键测试用例

### [TC-HK-01] 快捷键字符串合法性校验
- **描述**：验证合法与非法快捷键字符串在绑定前校验行为。
- **测试步骤**：
  1. 输入 `CTRL+SHIFT+V`。
  2. 输入 `NOT_A_SHORTCUT`。
- **预期结果**：
  - 步骤 1：校验通过。
  - 步骤 2：返回非法快捷键错误。

### [TC-HK-02] 重绑前清理旧绑定
- **描述**：每次绑定前应清理旧快捷键，防止重复注册冲突。
- **测试步骤**：
  1. 先绑定 `CTRL+SHIFT+V` / `CTRL+SHIFT+M`。
  2. 再绑定 `ALT+V` / `ALT+M`。
- **预期结果**：
  - 第二次绑定前调用 `unregister_all()`。
  - 运行时状态保存为最新快捷键组合。

### [TC-HK-03] 解绑快捷键
- **描述**：解绑后系统不应保留快捷键配置。
- **测试步骤**：
  1. 执行绑定。
  2. 调用解绑命令。
- **预期结果**：
  - 调用 `unregister_all()` 成功。
  - `get_global_hotkeys()` 返回空值。
